## Introduction
Shadow Kingdom Battle Game is a Python project built to demonstrate object‑oriented programming through an interactive, turn‑based combat system. Players choose from four character classes—Warrior, Mage, Archer, or Paladin—each with unique stats, abilities, and dodge mechanics. The goal is to defeat the Evil Wizard using strategy, healing, and special abilities.

The project highlights inheritance, polymorphism, method overriding, and clean class design. The battle loop includes dodge logic, random enemy actions, and regeneration mechanics to create a dynamic gameplay experience.

## Features
- Four playable character classes with unique abilities  
- Evil Wizard enemy with multiple attack types  
- Dodge system with class‑specific dodge chances  
- Healing abilities for all classes  
- Randomized enemy actions for replayability  
- Clean OOP structure using inheritance and polymorphism  
- Turn‑based combat with user input  
- Regeneration and special attack mechanics

# Gameplay Overview
Attack: Perform your class’s main attack.

Special Ability: Activate a dodge‑based defensive move.

Heal: Restore health (amount varies by class).

View Stats: Check your current health and attack power.

Wizard Turn: The Evil Wizard performs a random attack and regenerates health.
